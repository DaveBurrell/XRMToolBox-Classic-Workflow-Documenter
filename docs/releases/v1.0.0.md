# v1.0.0 Release Notes

Release date: 2026-03-16

## Highlights

- First production-ready release of the Classic Workflow Documenter.
- Supports Dynamics 365 unmanaged solution ZIP parsing for classic workflows.
- Produces business-friendly DOCX reports, overview documents, and diagrams.
- Includes CLI and desktop (WPF) experiences.

## Included Features

- Extraction and normalization of classic workflow definitions.
- Trigger and execution metadata surfaced in generated artifacts.
- Quality/risk analysis and dependency mapping.
- Per-workflow DOCX and solution/portfolio overview DOCX generation.
- Step inventory CSV output for audit and downstream processing.
- SVG diagram export and DOCX-embedded PNG diagrams.
- Enhanced diagram visuals:
  - Action-type color coding.
  - Semantic node shapes for decision/action/terminal semantics.
  - Vertical flow layout with large-diagram splitting.
  - Business-friendly split view naming.
- Diagram detail control:
  - Standard for concise views.
  - Detailed for richer metadata and traceability.

## Runtime Modes

- CLI: extract, document, docx, pack, batch.
- WPF: configure-driven single and batch generation with progress and cancellation support.

## Validation Snapshot

- Build status: passing.
- Core tests: 37 passed.
- WPF tests: 2 passed.

## Known Limitations (v1.0.0)

- Scope is classic workflow process documentation from unmanaged solution packages.
- No direct Dataverse live-connection mode in v1.
- No PDF export in v1.
- No runtime execution telemetry in v1.

## Upgrade and Compatibility

- Baseline release; no migration steps required.
- Recommended .NET SDKs should match project targets defined in solution project files.
